# SnakeRPG 밸런스/수치 문서

본 문서는 코드 및 씬 설정을 기준으로 한 밸런스/수치 정의서입니다.
기준일: 2026-02-25

---

**1) 플레이어 기본 스탯 (씬 기준)**
- HP: 5
- Max HP: 5
- Attack: 1
- Move Speed (base): 2

**2) 성장/레벨업**
- 경험치 테이블 (코드 기준)
  - Lv1 → Lv2: 100
  - Lv2 → Lv3: 200
  - Lv3 → Lv4: 400
  - Lv4 → Lv5: 800
  - Lv5 → Lv6: 1500
- 레벨업 보상 카드
  - 공격력 +1/+2/+3
  - 최대 HP +1/+2/+3 (즉시 회복 포함)
  - 즉시 회복 +1/+2/+3

**3) 이동 속도 증가 규칙**
- 생존 시간 10초마다 `baseMoveSpeed * 0.1`만큼 증가

**4) 몬스터 전투 계산 규칙**
- hitsNeeded = ceil(monsterHP / playerATK)
- damageToPlayer = (hitsNeeded - 1) * monsterDamage
- playerHP > damageToPlayer 일 때만 승리

**5) 스폰 주기 (씬 기준)**
- 몬스터: 10s
- 아군: 10s
- 음식: 10s

**6) 맵 확장 규칙**
- 레벨 4마다 맵 1칸 확장 (가로/세로 번갈아 증가)
- 최대 크기: 18x18

## MonsterBalance.csv (시간 기반 스탯 테이블)

현재 CSV는 헤더가 깨져 있어, 아래는 정리된 권장 포맷입니다.

권장 헤더:
```
Time,FinalHP,FinalDamage
```

현재 데이터(정리본):
```
0,1,0
10,1,0
20,1,0
30,1,0
40,1,0
50,2,0
60,2,0
70,3,0
80,3,0
90,4,0
100,4,0
110,5,0
120,5,0
130,6,0
140,6,0
150,7,0
```

주의
- 코드에서는 CSV 값을 “덮어쓰기”로 적용
- FinalHP가 0 이하일 경우 최소 1로 보정
- FinalDamage가 0이면 실제 공격력도 0

## 추가 제안(밸런스 개선안)
- 중반 이후 난이도 상승 체감 강화: 60초 이후 공격력도 1 이상으로 점진 상승
- 스폰 간격 분리: 몬스터 6~8초, 음식 12~15초, 아군 20초
- 자동 모드 전용 보정: 몬스터 HP 상승치를 소폭 낮추거나, 회복 카드 확률 증가
