# SnakeRPG 아트/사운드 가이드라인

본 문서는 현재 프로젝트의 자산 구성과 UI/게임 톤을 기준으로 한 기본 아트/사운드 방향성 문서입니다.

---

**1) 아트 스타일**
- 시점: 2D 탑다운, 타일 기반
- 톤: 미니멀/아케이드, 명확한 가독성 중심
- 스케일: 타일 크기와 Sprite 크기의 자동 보정(코드 기준)
  - `tileSize` 기준 스케일 조정
  - 아이템/몬스터 `itemSizeRatio` 적용

**2) 컬러 가이드**
- 바닥: 어두운 그레이 톤 2종
  - `floorColor1: (0.2,0.2,0.2)`
  - `floorColor2: (0.15,0.15,0.15)`
- 경계/벽: 중간 회색
  - `borderColor: (0.5,0.5,0.5)`
- UI: 흰색/검정 텍스트 기반 + 강조색(노랑, 초록, 빨강)

**3) 타이포그래피**
- 폰트: NotoSansKR SDF 계열
- UI 텍스트 크기/굵기 일관성 유지
- 전투/획득 텍스트는 짧고 직관적 (예: `HP+1`, `G+10`)

**4) VFX 가이드**
- 고정형 이펙트: 공격/사망/피격/섭취/아군 합류
- 추적형 이펙트: 레벨업
- 유지 시간: 애니메이션 길이나 파티클 duration을 기준으로 자동 삭제
- 과도한 화면 점유 방지: 1회 이펙트당 1~2초 내 소멸 권장

**5) SFX 가이드**
- 재생 방식: `PlayOneShot`으로 중첩 재생 허용
- 기본 세트
  - 공격, 사망, 섭취, 아군 합류, 레벨업, 피격, 게임오버
- 권장 톤
  - 공격/피격: 짧고 날카로운 타격음
  - 섭취/합류: 밝고 경쾌한 피드백
  - 레벨업: 상승/보상 느낌
  - 게임오버: 짧고 낮은 톤

**6) UI 애니메이션 (권장)**
- 레벨업 패널: 페이드 + 스케일 인
- 알림 텍스트: 2초 유지 후 페이드 아웃
- 플로팅 텍스트: 상향 이동 + 페이드
